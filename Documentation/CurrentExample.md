# Current Example: SmartContentDiet UII Migration Bridge

This branch demonstrates the bridge plugin **SmartContentDiet UII Migration Bridge** in Unreal Engine 5.4.

## What this plugin does

Interface-only bridge that lets SmartContentDiet surface migration and upgrade-impact advice from Unreal Integration Intelligence handoff artifacts.

## First Run

1. Open TinyToolDev_Examples.uproject in Unreal Engine 5.4.
2. Ensure Plugins/SmartContentDietUIIMigrationBridge is present in this repository branch.
3. Enable the plugin in the project plugin settings if required by your project settings.
4. Build the project editor target for Win64 Development and follow your plugin-specific tests/docs if available.

## Required dependencies

- SmartContentDiet
- UnrealIntegrationIntelligence
